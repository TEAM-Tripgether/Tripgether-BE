## [0.0.3] - 2025-10-06

**New Features**
- Spring Boot 앱 초기화, 예제 API(생성/조회) 추가

**Documentation**
- README 대폭 확장, 버전 표기 업데이트, 가이드 문서 제거

**Tests**
- 기본 컨텍스트 로드 테스트 추가

**Chores**
